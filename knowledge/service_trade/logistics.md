# 物流场景收款 — 专项指南

## 场景描述

中国物流/货代企业为东南亚客户提供跨境物流服务（海运/空运/陆运），需要收取运费/货代费。

## 收款痛点

| 痛点 | 具体表现 |
|------|---------|
| 币种复杂 | 不同国家客户付不同币种（THB/MYR/PHP/IDR/VND/USD） |
| 到账慢 | 银行电汇 T+3~5，影响资金周转 |
| 汇损高 | 中转行扣费 + 汇率差 |
| 对账难 | 多币种、多笔、多客户，财务对账耗时 |
| 退汇风险 | 单据不全导致退汇，客户体验差 |

## Ksher 解决方案

```
东南亚客户（付本地币种）
   ↓
Ksher 本地账户收款（THB/MYR/PHP/IDR）
   ↓
结汇为人民币 / 保留外币
   ↓
提现至物流企业国内账户
```

## 所需材料

1. 物流服务合同
2. 商业发票（Commercial Invoice）
3. 运输单据（提单/空运单/陆运单）

## 特殊优势

- **本地币种收款**：泰国客户付泰铢、马来客户付马币，客户付款更方便
- **多币种归集**：一个 Ksher 账户管理多国收款
- **API 对接**：支持 ERP/WMS 系统对接，自动对账

## 费率

同跨境货贸定价：标准 0.40%，分层最低 0.05%。

## 案例话术

> "深圳某货代公司，月收东南亚运费约 200 万人民币，之前用银行电汇，每笔固定费 150 元 + 1.5% 手续费 + 汇损。切换到 Ksher 后，费率降到 0.2%，一年省了 8 万多。"
