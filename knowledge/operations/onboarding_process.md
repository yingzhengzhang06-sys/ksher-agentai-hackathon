# 开户流程指南

## 开户前准备

### 企业客户所需材料

| 材料 | 说明 | 备注 |
|------|------|------|
| 营业执照 | 中国大陆企业营业执照 | 彩色扫描件 |
| 法人身份证 | 正反面 | 彩色扫描件 |
| 银行开户许可证 | 或基本存款账户信息 | — |
| 贸易证明 | 电商平台店铺链接/合同/发票 | 证明真实贸易背景 |
| 股东结构 | 持股 25% 以上股东信息 | 反洗钱要求 |

### 个人客户所需材料

| 材料 | 说明 |
|------|------|
| 身份证 | 正反面 |
| 银行卡 | 用于提现 |
| 贸易证明 | 店铺链接/交易记录 |

## 开户流程

```
Day 1: 提交资料 → Ksher 初审（1 个工作日）
Day 2: 补充材料（如需）→ 终审（1 个工作日）
Day 3: 账户开通 → 获取收款账户信息
Day 4: 测试收款 → 确认到账正常
Day 5: 正式使用
```

**总耗时：3-5 个工作日**

## 常见问题

**Q：开户需要费用吗？**
A：Ksher 开户免费，无账户管理费。

**Q：没有海外公司能开户吗？**
A：可以。中国大陆公司/个人即可开通 Ksher 账户。

**Q：审核不通过的原因？**
A：常见原因：材料不清晰、贸易背景不真实、涉及敏感行业。

## 代理商协助开户

代理商可以帮助客户：
1. 预审材料完整性
2. 指导材料准备
3. 跟进审核进度
4. 解答开户问题
