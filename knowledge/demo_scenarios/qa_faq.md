# 评审团 Q&A 问题库

## 产品技术类

**Q1：Ksher 和 PingPong/万里汇的核心区别是什么？**
> A：三个核心差异：① Ksher 在东南亚 5 国持有央行支付牌照，资金本地清算不用中转；② 到账快一天（T+1 vs T+2）；③ 汇率更优（点差 0.3% vs 0.5%）。PingPong 和万里汇是做欧美起家的，东南亚没有本地牌照。

**Q2：资金安全怎么保障？**
> A：Ksher 在泰国、马来西亚、菲律宾、印尼、香港均持有央行/金管局颁发的支付牌照，受当地金融监管。客户资金与运营资金隔离存放，每一笔交易都可追溯。

**Q3：费率 0.05% 是最低吗？什么条件能拿到？**
> A：0.05% 是 S 级客户（月流水 ≥50 万美金）的分层定价。标准定价（D 级）是 B2C 0.8%、B2B 0.4%。具体费率根据客户实际交易量和谈判确定。

**Q4：开户需要多久？需要什么材料？**
> A：企业开户 3-5 个工作日。需要营业执照、法人身份证、贸易证明（店铺链接/合同）。全程线上完成，无需赴港或赴当地。

## 商业模式类

**Q5：这个 AgentAI 怎么帮代理商赚钱？**
> A：三个层面：① 降低培训成本——新代理商 1 天上手，不用背 40 多页产品手册；② 提升成交率——AI 生成的销售话术基于真实数据，说服力更强；③ 节省时间——3 分钟生成完整作战包，以前要准备 2 小时。

**Q6：和 ChatGPT 有什么区别？**
> A：三个核心差异：① 专属知识库——40+ 文档的 Ksher 产品数据，ChatGPT 不知道泰国 B2B 费率是 0.4%；② 战场自动适配——输入客户信息自动判断增量/存量/教育战场，生成不同策略；③ 数据闭环——每次拜访结果反馈给系统，AI 越用越准。

**Q7：代理商会不会依赖 AI，自己不动脑子了？**
> A：AgentAI 是"副驾驶"不是"自动驾驶"。AI 生成话术和方案，但最终的沟通策略、客户关系维护、谈判节奏还是靠代理商自己。AI 解决的是"信息准备"问题，让代理商把精力放在"客户互动"上。

## 技术实现类

**Q8：为什么用 Kimi + Claude 双模型？**
> A：分工明确：Kimi K2.5（创意型）负责话术/内容/设计——中文创作能力强；Claude Sonnet 4.6（精准型）负责成本计算/方案/知识问答——逻辑严谨、数据准确。双模型组合发挥各自优势。

**Q9：知识库怎么维护？产品更新了怎么办？**
003e A：知识库是本地 Markdown 文件，更新方式：① 渠道负责人直接在管理后台编辑；② 保存后即时生效，下次 AI 调用自动读取最新内容；③ V2 版本将接入 Git 版本管理，可回滚。不需要重新训练模型。

**Q10：API 成本高吗？代理商用得起吗？**
003e A：单次作战包生成成本约 0.3-0.5 美元（调用 4 个 Agent）。按每个代理商每天生成 3 个作战包计算，月成本约 30-50 美元。相比代理商多成交一个客户带来的收益，边际成本极低。

## 市场定位类

**Q11：目标用户是谁？市场有多大？**
003e A：第一阶段目标用户是 Ksher 的城市代理商（目前已覆盖 20+ 城市）。中国有 200 万+ 跨境电商卖家，其中 30%+ 做东南亚市场。按每个城市 50 个活跃代理商、月费 500 元计算，年市场规模 600 万+。

**Q12：竞品有没有类似产品？**
003e A：目前跨境支付行业还没有面向代理商的 AI 销售助手。竞品（PingPong/万里汇/XTransfer）主要提供卖家端工具，没有代理商赋能产品。这是我们的差异化壁垒。

**Q13：V1 和 V2 的规划是什么？**
003e A：V1（Demo 版）：7 个 Agent + 一键备战 + 本地知识库注入。V2（生产版）：向量数据库 RAG + 数据闭环学习 + 更多独立工具（内容工厂/设计工作室）。V3（生态版）：代理商社区 + 最佳实践共享 + 自动化工作流。
