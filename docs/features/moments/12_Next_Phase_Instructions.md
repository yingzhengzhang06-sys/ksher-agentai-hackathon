## 3. ARCH-AI-GOV-01 指令

### 目标

建立真实 AI 的上线治理能力。

### 必须覆盖

- 真实 AI 输出质量抽样标准。
- 成本监控与单次调用成本记录。
- 超时、失败、空响应、格式异常的兜底策略。
- 频控策略与滥用防护。
- Mock / 真实模式切换说明。
- 日志脱敏规则。
- 生产环境密钥管理边界。

### 禁止事项

- 不读取或输出 API Key、Token、`.env`。
- 不直接部署生产。
- 不接入真实微信接口。
- 不扩大到素材库、CRM、数据分析。

### 输出物

- `docs/features/moments/21_AI_Governance_Plan.md`
- 必要的工程任务拆解。
- 必要的 QA 验收标准。

## 4. DEPLOY-MOMENTS-01 指令

### 目标

明确发朋友圈数字员工的演示 / 测试部署方式。

### 必须覆盖

- Streamlit 启动方式。
- FastAPI 启动方式。
- 端口配置。
- 局域网访问方式。
- 健康检查。
- 日志路径。
- 失败排查步骤。

### 禁止事项

- 不直接部署生产。
- 不修改密钥或生产配置。
- 不把本地临时环境当作正式生产环境。

### 输出物

- `docs/features/moments/22_Deploy_Runbook.md`
- 如需脚本，另开工程任务评审后再实现。

- 不使用 `git add .`。

### 输出物

1. `WORKTREE-CLEANUP-01`：先降低仓库污染风险。
