# 战场类型与销售策略

## 三种战场类型

### 1. 增量战场（从银行抢客户）

**客户特征**：
- 当前使用银行电汇收款
- 对第三方支付了解有限
- 年流水通常 500 万以上
- 行业：传统外贸、B2B 大宗贸易

**核心策略**："帮你算一笔你没算过的账"

**切入点**：
1. 隐性成本唤醒：把银行电汇的 5 项成本列出来
2. 对比冲击：一年能多省多少钱
3. 降低门槛：不用关银行账户，先并行试用

**常见异议**：
- "没听过 Ksher" → 展示牌照资质
- "安全吗" → 强调持牌 + 资金隔离
- "嫌麻烦" → 开户简单，3-5 个工作日

### 2. 存量战场（从竞品抢客户）

**客户特征**：
- 当前使用 PingPong/万里汇/XTransfer 等
- 对跨境支付有了解
- 年流水 100-1000 万
- 行业：跨境电商、多平台卖家

**核心策略**："东南亚这块，我们有硬差异"

**切入点**：
1. 东南亚本地化：本地牌照 + 本地账户 + 本地客服
2. 到账速度：T+1 vs T+2
3. 费率优势：分层定价，量大更省

**常见异议**：
- "已经在用 XX 了" → 不用换，并行对比
- "费率差不多" → 算总成本（汇率+到账时间+服务）
- "懒得换" → 切换成本低，开户免费

### 3. 教育战场（新客户/未选定渠道）

**客户特征**：
- 刚开始做东南亚市场
- 对跨境收款完全不了解
- 没有固定收款渠道
- 行业：新入局卖家、品牌出海

**核心策略**："跨境收款有 3 个坑，你知道几个"

**切入点**：
1. 教育先行：讲清跨境收款的 5 项成本
2. 建立标准：教客户怎么选收款渠道
3. 引导选择：自然导向 Ksher 的优势

**常见异议**：
- "不着急" → 早布局早省钱
- "再看看" → 先发资料，保持联系
- "先了解一下" → 提供知识库/案例

## 战场判断方法

根据客户当前收款渠道自动判断：

```python
if current_channel in ["银行", "电汇", "TT"]:
    battlefield = "increment"      # 增量战场
elif current_channel in ["PingPong", "万里汇", "XTransfer", "连连"]:
    battlefield = "stock"          # 存量战场
else:
    battlefield = "education"      # 教育战场
```
